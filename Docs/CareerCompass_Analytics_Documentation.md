# CareerCompass Analytics Documentation

This document lists the KPIs and analytics implemented across the six CareerCompass Power BI dashboard pages.

## Page 1 — Student Analytics Dashboard

1. **Total Students** — Finds the total number of students.
2. **Average Readiness** — Finds the average placement readiness score.
3. **Average CGPA** — Finds the average CGPA of students.
4. **Placement Ready Students** — Finds the total number of placement-ready students.
5. **Placement Status Distribution** — Shows students by placement readiness status.
6. **Students by Branch** — Finds the total number of students in each branch.
7. **Average CGPA by Branch** — Finds the average CGPA for each branch.

## Page 2 — Coding & SQL Analytics Dashboard

1. **Average Problems Solved** — Finds the average number of LeetCode problems solved.
2. **Average Contest Rating** — Finds the average LeetCode contest rating.
3. **Average SQL Assessment** — Finds the average SQL assessment score.
4. **Average SQL Practice** — Finds the average SQL practice score.
5. **Average Contest Rating by Branch** — Finds the average contest rating for each branch.
6. **SQL Assessment vs Contest Rating** — Compares SQL assessment scores with contest ratings.
7. **Average SQL Assessment by Branch** — Finds the average SQL assessment score for each branch.
8. **LeetCode Difficulty Distribution** — Shows solved problems by Easy, Medium, and Hard difficulty.

## Page 3 — Project & GitHub Analytics Dashboard

1. **Average Project Score** — Finds the average project score.
2. **Average Project Completion** — Finds the average project completion percentage.
3. **Average GitHub Commits** — Finds the average number of GitHub commits.
4. **Total Projects** — Finds the total number of student projects.
5. **Student Project Details** — Shows each student's projects and project scores.
6. **GitHub Commits vs Project Score** — Compares GitHub commit activity with project scores.
7. **Average Project Score by Branch** — Finds the average project score for each branch.
8. **GitHub Primary Language Distribution** — Shows the distribution of primary GitHub languages.

## Page 4 — Placement Readiness Analytics Dashboard

1. **Average Readiness Score** — Finds the average overall readiness score.
2. **Average LeetCode Score** — Finds the average LeetCode readiness score.
3. **Average SQL Score** — Finds the average SQL readiness score.
4. **Placement Ready Students** — Finds the total number of placement-ready students.
5. **Above Average Readiness Students by Branch** — Finds students scoring above the overall readiness average by branch.
6. **Student Readiness Rankings** — Ranks students by overall readiness score.
7. **Readiness Component Scores by Branch** — Compares Project, Mock Test, LeetCode, and SQL scores by branch.
8. **Placement Readiness Distribution** — Shows students by placement readiness category.

## Page 5 — Mock Test Performance Analytics Dashboard

1. **Average Mock Test Score** — Finds the average mock test score.
2. **Highest Mock Test Score** — Finds the highest mock test score.
3. **Total Mock Tests** — Finds the total number of mock test records.
4. **Average Readiness Mock Score** — Finds the average mock test readiness component score.
5. **Average Mock Test Score by Difficulty** — Finds the average score for each test difficulty.
6. **Average Mock Test Score Trend** — Shows the average mock test score over time.
7. **Above Average Mock Test Students by Branch** — Finds above-average mock test performers by branch.
8. **Student Mock Test Rankings** — Ranks students by average mock test score.

## Page 6 — Placement & Company Applications Analytics Dashboard

1. **Total Applications** — Finds the total number of company applications.
2. **Selected Students** — Finds the total number of selected application records.
3. **Selection Rate** — Calculates the percentage of selected applications.
4. **Average Package LPA** — Finds the average company package.
5. **Application Pipeline by Status** — Shows application records by application status.
6. **Applications by Company** — Finds the number of applications for each company.
7. **Application Status Distribution by Branch** — Shows application status percentages for each branch.
8. **Average Package LPA by Company** — Finds the average package offered by each company.
